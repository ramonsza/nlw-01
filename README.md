# Ecoleta NLW#01
 
Um projeto com foco em sustentabilidade, desenvolvido com as tecnologias: NodeJS, React e React-native

## Instalação

Para adquirir os módulos necessários execute, dentro de cada diretório do projeto, o comando a seguir:

```bash
npm install
```
